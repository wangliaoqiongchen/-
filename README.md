# 🛠️ 17合1手机端效率工具箱
**极简商务风 | 纯前端实现 | 无需安装APP**，集成文本处理、计算工具、单位换算、生活工具、编码工具五大类共17个实用功能，手机浏览器直接运行，适合日常办公、学习、生活场景。

## 📋 17个功能清单
| 分类         | 功能列表                                                                 |
|--------------|--------------------------------------------------------------------------|
| 文本处理     | 1.AI文本总结  2.字数统计  3.转大写  4.转小写  5.文本去重  6.文本反转     |
| 文本提取     | 7.提取数字  8.提取字母                                                   |
| 计算工具     | 9.随机数生成  10.大小写金额转换                                          |
| 单位换算     | 11.长度换算  12.重量换算  13.汇率换算（人民币↔美元）                    |
| 生活工具     | 14.倒计时  15.简易备忘录（本地存储）                                    |
| 编码工具     | 16.二维码生成  17.颜色代码转换（RGB→16进制）                            |

## ✨ 核心亮点
1. **轻量化**：纯HTML+CSS+JS开发，无后端依赖，打开即用
2. **适配手机**：按钮大小、输入框间距贴合触屏操作，横屏竖屏均兼容
3. **极简UI**：商务蓝主调+卡片式布局，无花哨特效，专注实用性
4. **本地存储**：备忘录功能基于localStorage，数据不丢失

## 📱 运行方式
1. 将3个文件放入手机同一文件夹
2. 用浏览器打开 `index.html` 即可使用
3. 支持添加到手机桌面，当作离线APP
